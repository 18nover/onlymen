Installieren von Tails

Windows

macOS

Linux

Console

Debian oder Ubuntu über die Befehlszeile und GnuPG

Tails nur herunterladen (für USB-Sticks)

Latest version: Tails 7.7.2 from 2026-05-04

Wenn Sie jemanden kennen, dem Sie vertrauen und der Tails bereits verwendet, können Sie Ihr Tails installieren, indem Sie dessen Tails klonen:

  • Installation durch Klonen von einem anderen Tails auf PC
  • Installation durch Klonen von einem anderen Tails auf Mac

Tails funktioniert nicht auf Smartphones oder Tablets. Die Hardware von Smartphones und Tablets unterscheidet sich erheblich von der Hardware von Computern. Derzeit ist es unmöglich, die Hardware von Smartphones und Tablets mit Linux-Distributionen wie Tails zum Laufen zu bringen.

System requirements and recommended USB sticks
