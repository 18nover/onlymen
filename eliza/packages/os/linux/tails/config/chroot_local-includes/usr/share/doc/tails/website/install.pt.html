Instale Tails

Windows

macOS

Linux

Console

Debian ou Ubuntu usando a linha de comando e GnuPG

Download Tails only (for USB sticks)

Latest version: Tails 7.7.2 from 2026-05-04

Se você conhece alguém que já usa Tails e você confia nessa pessoa, você pode instalar o seu Tails usando o dela:

  • Instalar clonando de um outro Tails em um PC
  • Instalar clonando de um outro Tails em um Mac

Tails não funciona em smartphones ou tablets. O hardware de smartphones e tablets é muito diferente do hardware de computadores. Por enquanto, não é possível fazer smartphones e tablets funcionarem com distribuições Linux tais como Tails.
