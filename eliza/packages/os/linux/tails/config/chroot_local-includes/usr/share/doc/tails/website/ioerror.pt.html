Erro ao ler dados do pendrive USB com Tails

Esta página se aplica em 2 situações:

  • If you get the following error message when starting Tails:

    Error reading data from your Tails USB stick.
Please go to https://tails.net/ioerror for troubleshooting.

  • If you get the following notification while using Tails:

    Erro ao ler dados do pendrive USB com Tails

O hardware do seu pendrive provavelmente está falhando.

Recomendamos que você instale o Tails em um pendrive diferente
e recupere os dados do seu Armazenamento Persistente o mais rápido possível.

Esses alertas são exibidos quando o Tails detecta um problema ao ler ou gravar no seu pendrive USB. Muito provavelmente, esses alertas significam que o hardware do seu pendrive USB está falhando e pode quebrar completamente em breve.

Você deve parar de usar este pendrive USB para Tails e instalar um novo.

  • Se você não tiver um Armazenamento Persistente neste pendrive USB, recomendamos que você reinstale o Tails em um novo pendrive USB.

  • Se você tiver um Armazenamento Persistente neste pendrive USB, recomendamos que você pare de usar este Tails imediatamente, reinstale o Tails em um novo pendrive USB, e recupere os dados do seu Armazenamento Persistente o mais rápido possível.

    Para fazer isso, siga nossas instruções sobre como recuperar o Armazenamento Persistente de um Tails quebrado .
